COMO INSTALAR EM 30 MINUTOS
Você pode rodar no seu Mac.
Ou aderir à febre e comprar um Mac Mini só pra ele.
Mas tem uma alternativa mais simples e mais barata: VPS.
Por que usar um VPS?
→ Roda 24/7 sem depender do seu computador 
→ Pode te mandar mensagens proativamente 
→ Custa ~$5/mês na Hetzner
O que você precisa?
→ Conta na Hetzner (ou outro VPS com Ubuntu) 
→ API key da Anthropic (~$20-50/mês uso moderado) 
→ Telegram (recomendado) ou WhatsApp
Passo 1: Criar servidor na Hetzner
→ Cria conta em hetzner.com/cloud 
→ Novo servidor Ubuntu 24.04 
→ Plano CX22 ($4/mês, 4GB RAM) pra chat básico 
→ Plano CX32 ($7/mês, 8GB RAM) se quiser browser automation
Passo 2: Conectar via SSH
bash
ssh root@<seu-ip>
Criar usuário dedicado:
bash
adduser clawdbot
usermod -aG sudo clawdbot
su - clawdbot
Passo 3: Instalar Node.js
bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
source ~/.bashrc
nvm install 22
corepack enable pnpm
Passo 4: Instalar ClawdBot
Opção simples (recomendada):
bash
curl -fsSL https://clawd.bot/install.sh | bash
Ou from source:
bash
git clone https://github.com/clawdbot/clawdbot.git
cd clawdbot
pnpm install
pnpm build
pnpm ui:build
Passo 5: Rodar o wizard
bash
clawdbot onboard --install-daemon
O wizard vai te guiar:
→ Gateway: seleciona "local" 
→ Runtime: escolhe "Node" 
→ Auth: API key da Anthropic (Claude Opus 4.5 recomendado) 
→ Provider: Telegram (mais fácil que WhatsApp)
Pro Telegram: cria um bot em @BotFather, manda /newbot e pega o token.
Passo 6: Aprovar seu usuário
Manda uma mensagem pro bot no Telegram.
Depois aprova com:
bash
clawdbot pairing list telegram
clawdbot pairing approve telegram <código>
Pronto. Seu assistente está rodando 24/7.
