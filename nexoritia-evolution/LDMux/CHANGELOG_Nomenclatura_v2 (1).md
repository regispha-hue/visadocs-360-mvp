## 💡 NOTAS IMPORTANTES

### 1. Distinção LDM vs LDMux-OS

**Sempre manter claro:**
- **LDM** = Livro (O Livro dos Montes, a obra literária)
- **LDMux-OS** = Sistema (o operating system para criar livros)

**Analogia:**
- **Linux kernel** = Sistema operacional
- **Ubuntu, Debian, etc.** = Distribuições que usam Linux
- **LDMux-OS** = Sistema operacional literário
- **O Livro dos Montes** = Primeira obra criada com LDMux-OS

### 2. Filosofia do Nome

**"Mux" tem duplo significado:**
- **Unix:** Herança técnica (sistemas operacionais)
- **Multiplex:** Múltiplas vozes, múltiplas IAs, multiplicidade

**Perfeito para EA:** Sistema que multiplexa canais (humano + IA + liminar)

### 3. Pronuncia

**PT-BR:**

- "eldêmuks" (casual)

**EN:**
