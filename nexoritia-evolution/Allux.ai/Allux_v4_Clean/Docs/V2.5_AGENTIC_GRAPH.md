## ⚡ **NOVIDADES v2.5**

### **1. Agentic RAG - Roteador Autônomo**

**O que é:** Sistema que DECIDE sozinho onde buscar contexto.

**Como funciona:**
```python
# Usuário: "Escreva cena do Monte II"

# Agente analisa e decide:
decision = {
  "sources_to_query": ["kernel", "canon"],
  "reasoning": "Preciso das leis do Monte II + cenas similares",
  "confidence": 0.9
}

# Agente reúne apenas contexto necessário:
context = {
  "kernel": ["Ninguém respira no Monte II"],
  "canon": [fragmentos de cenas aprovadas]
}

# Agente gera texto com contexto correto
```

**Por que importa:**
- ✅ Economia de tokens (não consulta tudo)
- ✅ Contexto preciso (só o necessário)
- ✅ Decisão autônoma (não precisa especificar)

---

### **2. GraphRAG - Grafo de Conhecimento**

**O que é:** Mapa de entidades e relações do LDM.

**Estrutura:**
```
Nós (Entidades):
- Monte II
- Cinzel
- Apneia
- Personagens

Arestas (Relações):
- Cinzel --[DEFINE]--> Atômico
- Cinzel --[PERTENCE_A]--> Monte II
- Apneia --[RELACIONA_COM]--> Silêncio
```

**Como resolve o problema:**
```
ANTES (RAG comum):
Query: "Gume"
→ Busca palavra "Gume"
→ Não encontra "Cinzel" (sinônimo)

DEPOIS (GraphRAG):
Query: "Gume"
→ Busca nó "Gume"
→ Segue aresta [SINONIMO_DE] → "Cinzel"
→ Retorna contexto completo
```

---

### **3. Pipeline de Ingestão - ZIP 700p**

**O que é:** Script que processa rascunhos originais.

**Execução:**
```bash
python ingest_legacy.py /path/to/rascunhos_700p.zip
```

**O que faz:**
1. ✅ Extrai textos → Sources Vault (type=legacy)
2. ✅ Identifica entidades → Knowledge Graph (nós)
3. ✅ Detecta relações → Knowledge Graph (arestas)
4. ✅ Extrai fragmentos → RAG Canônico

**Resultado:**
```
=== INGESTÃO COMPLETA ===
Arquivos processados: 87
Fontes criadas: 87
Nós (entidades): 245
Arestas (relações): 412
Fragmentos extraídos: 523
```
